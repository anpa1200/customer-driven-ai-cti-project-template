# Customer-Driven AI CTI Project Template

A Docusaurus-ready version of the Customer-Driven AI CTI Project Template: a practitioner methodology for moving from intelligence requirements to validated hunts, detections, SOC handoff, and customer-facing delivery artifacts.

## Start Here

- [Part 1: Foundations](/docs/project-template/foundations) defines the analytic standards, governance model, scoring model, roles, artifacts, and detection-readiness concepts.
- [Part 2A: Phase-by-Phase Execution Guide](/docs/project-template/phase-by-phase-execution-guide) provides the delivery playbook from project charter through final reporting.
- [Part 2B: Reference Toolkit](/docs/project-template/reference-toolkit) contains AI workflows, task cards, quality gates, registers, a worked example, and delivery-package guidance.

## Intended Use

Use this repository as a publishable documentation site, a customer-project starting point, or a version-controlled methodology reference for CTI-to-detection-engineering work.
