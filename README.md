# Customer-Driven AI CTI Project Template

This repository contains a Docusaurus-ready documentation site for the Customer-Driven AI CTI Project Template.

## Local Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```

The generated static site is written to `build/`.

## GitHub Pages Deployment

```bash
USE_SSH=true npm run deploy
```

The Docusaurus config is set for `https://anpa1200.github.io/customer-driven-ai-cti-project-template/`.
